Principais formatações

Negrito / Destaque

Nessa frase tem uma palavra em negrito usando a tag <b> (Não semantica)
Nesta frase, tenho um termo em negrito usando a tag <strong> (Semantica)

Italico ou enfase

Nesta farse, temos um termo em italico, usando a tag <i> (Não semantica)

Nesta frase temos um termo em enfase, usando a tag <em> (Semantica)

Podemos também marcar um texto usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido, mas não considerado, basta usar a tag <del>

Texto inserido

Podemos marcar um texto como inserido, para dar uma enfase e indicar que indicar que ele foi adicionado depois, basta usar a tag <ins> (semantica)

Existe também o texto sublinhado, utilizando a tag <u> (não semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilize a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, utilize a tag <sub>
